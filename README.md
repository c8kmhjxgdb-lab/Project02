# 星愿之子 (Starchild2D)

> 每个大人的心里，都住着一个相信魔法的孩子。

## 🌟 这是什么？

一款 **2D 俯视角动作 RPG**，用 C++ 重新诠释童年最纯粹的幻想——  
在这里，你能发射火球、和公主建立羁绊、在程序生成的世界里冒险。  
没有复杂的数值，没有逼氪的商城，只有最原始的「好玩」。

## ✨ 核心体验

- **🔥 爽快战斗** — 火球术、冰锥术、飞踢、抓取投掷，技能组合自由释放
- **💖 情感羁绊** — 公主不是 NPC，是会记住你每一次对话的伙伴（好感度 0-1000）
- **🗺️ 无限世界** — 基于噪声算法生成的地图，每次冒险都是独一无二的
- **🎭 情绪系统** — 角色会难过、会生气， grievance 值满屏泛红的那种真实

## 🛠️ 技术栈

| 领域 | 技术 |
|------|------|
| 语言 | C++17 |
| 图形 | OpenGL 3.3 Core + SDF 渲染（零精灵纯数学绘制） |
| 物理 | Box2D v3（零重力俯视角） |
| 脚本 | Lua + sol2（对话树、技能配置全可热修改） |
| 构建 | CMake + vcpkg |

## 🎮 快速开始

```bash
# 前提：安装 vcpkg 并设置 VCPKG_ROOT

cmake -B build
cmake --build build --config Release

# 运行（必须在 build 目录下）
cd build && ./Release/Starchild2D.exe
```

### 操作方式

| 键位 | 功能 |
|------|------|
| WASD | 移动 |
| J | 火球术 |
| L | 冰锥术（附带减速） |
| 空格 | 冲刺（短暂无敌） |
| K | 抓取/投掷（怪力） |
| E | 互动（对话、发泄） |
| 鼠标滚轮 | 缩放视角 |

## 📁 项目结构

```
src/
├── Engine/        # 引擎层：渲染、物理、摄像机、LuaVM
├── Game/          # 游戏层：地图、AI、战斗、对话、情绪
└── Utils/         # 工具：数学库、着色器加载
assets/
├── shaders/       # GLSL 着色器（角色/弹幕/后处理全 SDF）
└── scripts/       # Lua 脚本（对话树、技能配置）
```

## 🏗️ 架构亮点

- **组件化对象池** — 弹幕、敌人、粒子全部预分配，零运行时内存抖动
- **类型安全实体 ID** — `ProjectileId`、`EnemyId` 不是裸整数，编译期防错
- **回调驱动解耦** — 受伤/死亡/对话/情绪变化全是事件，系统之间不硬依赖
- **Y 排序渲染** — 高装饰物（大树）和角色正确遮挡，俯视角也有深度感
