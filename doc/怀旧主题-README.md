## 核心概念

> **「童心 vs 长大」** 是核心矛盾。
> - 9 位公主 = 童心界的 9 个守护者，每个公主守护一种"童心能力"（玩/看/强/信/学/唱/燃/问/听）
> - 9 大 BOSS + 1 终 BOSS = 童年阴影的拟人化（充值大帝、黑屏大王、后母皇后、补习班校长等）
> - 9 大领域 = 童年的 9 种世界（4399 厅 / 动画厅 / 漫威基地 / 童话森林 / 校园 / 童谣村 / 漫画城 / 科学实验室 / 音乐厅）
> - 童心值 0-1000 是核心机制，归零 = 坏结局，9 位公主全救 + 通关 = 真结局

### 9 公主（按领域顺序）
1. **琪琪**（橙/游戏/琪琪·像素之心）— 守护"玩"

3. **艾琳**（红/超级英雄/艾琳·星盾）— 守护"强"
4. **贝儿**（粉/童话/贝儿·魔镜）— 守护"信"
5. **小雪**（白蓝/校园/小雪·铃声）— 守护"学"
6. **铃铛**（黄/童谣/铃铛·小燕子）— 守护"唱"

## 设计原则

1. **童心 vs 长大** 是核心矛盾
2. **公主 = 童心分身**（不是等待救援的弱者）
3. **BOSS = 童年阴影拟人化**（不是抽象的魔王）
4. **武器 = 童年玩具**（不用现实武器）
5. **音乐 = 童年耳朵**（每首 BGM 对应一段真实童年记忆）
6. **不允许**：付费抽卡、逼氪、剧透、暴力血腥
7. **不卖惨、不说教**
8. **AI 提示词要"可喂即用"**
