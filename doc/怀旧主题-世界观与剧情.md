# 《星愿之子》怀旧主题 — 世界观与剧情设计

> 重写目标：把怀旧元素从“名词堆叠”改成“剧情动机、关卡规则、战斗机制和可生成资产”。  
> 设计原则：不直接使用现实游戏、动画、品牌、角色的正版称呼；只提炼玩家熟悉的记忆体验，并转化为原创世界、原创敌人和原创玩法。

---

## 1. 核心方向

《星愿之子》的主题不是“把童年作品摆满地图”，而是：

```text
玩家在长大过程中失去的自由、好奇、想象和勇气，
会在童心界变成被封存的地图、被打断的音乐、被改写的结局和被锁住的玩法。
```

怀旧感不靠直接喊出某个作品名，而靠玩家一看就懂的处境：

- 免费小游戏突然被弹窗挡住。
- 电视节目正精彩时变成雪花屏。
- 课间十分钟被铃声和试卷吞掉。
- 漫画停在“下一话”。
- 实验失败后被要求只写标准答案。
- 长大以后，所有颜色都像被调低了饱和度。

这些记忆要在游戏中变成规则，而不是只变成台词。

## 2. 世界观

### 2.1 叠层世界

世界由两层重叠地图构成：

| 层级 | 名称 | 玩家看到的样子 | 玩法含义 |
|------|------|----------------|----------|
| 上层 | 现实层 | 小区、学校、商店、办公室、灰色街道 | 主线压力来源，童心值下降的原因 |
| 下层 | 童心层 | 游乐厅、放映厅、操场、童话森林、漫画城、实验室 | 主要探索、战斗、基地和公主羁绊发生地 |

现实层不是“坏世界”。它代表成长、责任和日常秩序。真正的反派是把成长变成单一答案的力量：它要求孩子提前变得无聊、功利、服从，并把想象力视为错误。

### 2.2 童心值

童心值是主角长期资源，范围 0-1000。它不是普通生命值，而是控制世界可见度、剧情分支和能力强度的核心系统。

| 数值区间 | 世界表现 | 玩法影响 |
|----------|----------|----------|
| 800-1000 | 颜色鲜明，隐藏物可见 | 公主羁绊充能加快，隐藏路线出现 |
| 500-799 | 正常状态 | 可释放大部分主动技能 |
| 200-499 | 画面轻微泛灰 | NPC 台词偏担心，部分玩具失去互动 |
| 1-199 | 低童心警报 | 移动下降，敌人“催熟”技能增强，基地任务强制推荐 |
| 0 | 童心断线 | 进入坏结局警告或强制回基地保护流程 |

童心值的主要来源：

- 完成章节核心任务。
- 击败领域 BOSS。
- 在秘密基地布置童年物件。
- 与公主完成羁绊事件。
- 修复被打断的游戏、音乐、故事或实验。

童心值的主要消耗：

- 被敌人命中。
- 选择“快速变强但失去童心”的诱导奖励。
- 长时间不回基地休息。
- 在剧情中屈服于“标准答案”“快点长大”“别玩了”等压制性规则。

### 2.3 童心碎片

九位公主是九种童心能力的守护者。她们不是等待救援的奖品，而是被封住的玩法维度。

| 公主 | 守护能力 | 玩法解锁方向 | 对应领域 |
|------|----------|--------------|----------|
| 琪琪 | 玩 | 弹幕、小游戏、像素机关、试玩币 | 弹窗游乐厅 |
| 露露 | 看 | 胶片修复、色彩切换、黑屏反制 | 雪花放映厅 |
| 艾琳 | 强 | 变身、护盾、救援目标、团队合击 | 纸皮英雄基地 |
| 贝儿 | 信 | 改写结局、选择分支、童话契约 | 反面童话森林 |
| 小雪 | 学 | 课间计时、错题转化、考试压力反制 | 铃声操场 |
| 铃铛 | 唱 | 节奏桥、旋律谜题、噪音净化 | 跑调童谣镇 |
| 美琪 | 燃 | 分镜移动、速度线冲刺、补完结局 | 断格漫画城 |
| 诺诺 | 问 | 元素实验、假设验证、非标准解法 | 问号实验室 |
| 乐乐 | 听 | 音准校正、合奏、回声路径 | 回声音乐厅 |

### 2.4 反派势力：灰钟局

反派组织名建议改为 **灰钟局**。它比“长大王国”更有原创性，也能承接时间、规训、灰色办公和催熟主题。

灰钟局的目标不是毁灭世界，而是把童心层全部折叠进现实层，让每个孩子只剩下一条“标准成长路径”。

| 反派职位 | 原创称呼 | 压制的童心 | 主要机制 |
|----------|----------|------------|----------|
| 第一执事 | 六元冠冕 | 玩 | 弹窗遮挡、诱导购买、假奖励 |
| 第二执事 | 静屏馆长 | 看 | 黑屏、雪花、缺帧、色彩剥夺 |
| 第三执事 | 塑料特效王 | 强 | 假英雄、廉价爆炸、救援对象替换 |
| 第四执事 | 删改皇后 | 信 | 篡改结局、反转契约、坏结局循环 |
| 第五执事 | 补课总监 | 学 | 试卷弹幕、铃声锁门、时间压迫 |
| 第六执事 | 噪声公爵 | 唱 | 节奏干扰、音符污染、控制反转 |
| 第七执事 | 断更书记 | 燃 | 分镜断裂、进度冻结、回忆循环 |
| 第八执事 | 标准答案博士 | 问 | 单答案陷阱、实验失败惩罚 |
| 隐藏执事 | 失谐琴师 | 听 | 音准偏移、合奏失衡、回声迷宫 |
| 终局首领 | 灰钟局长 | 全部 | 时间加速、颜色归零、选择收束 |

## 3. 主线结构

主线采用“八章主线 + 一章隐藏 + 终章”的结构。每章都必须满足四个条件：

1. 有一个清晰的童年记忆处境。
2. 这个处境必须变成一个独特玩法。
3. BOSS 机制必须体现该处境的压迫感。
4. 通关奖励必须改变玩家之后的行动方式。

### 序章：六一傍晚的星星糖

**核心记忆**：长大前最后一个还能理直气壮过六一的傍晚。  
**剧情目标**：主角星愿在小卖部外捡到一颗发光的星星糖，第一次看见现实街道下方重叠的童心层。  
**玩法目标**：教学移动、互动、童心值、基地入口和第一场小战斗。

关键剧情：

- 星愿发现现实中的旧游戏厅门面，在童心层变成“弹窗游乐厅”的入口。
- 琪琪从一堆卡住的窗口里跳出来，告诉星愿：这不是回忆展览，而是正在坏掉的童心系统。
- 灰钟局已经封住其他八位公主，童心层的九个领域正在被折叠。

机制设计：

- 第一次童心值下降时，画面饱和度降低。
- 玩家喝下“星星糖水”后，童心值恢复，隐藏门显形。
- 教学战斗的敌人不直接伤害生命，而是偷走屏幕颜色。

### 第一章：弹窗游乐厅

**核心记忆**：网页小游戏的快乐总被弹窗、加载条和“变强诱导”打断。  
**守护公主**：琪琪。  
**反派**：六元冠冕。  
**章节奖励**：像素手柄，解锁“试玩币机关”和基础弹幕反制。

玩法机制：

- 地图中会弹出遮挡窗口，玩家需要用攻击或移动路线关闭。
- 部分弹窗写着“立即变强”，点击会短暂提升攻击但扣童心值。
- 玩家收集“试玩币”开启隐藏小游戏门，完成小游戏可恢复童心。
- BOSS 会把场地切成多个小窗口，玩家必须在可视区域内找安全格。

BOSS 机制：

| 阶段 | 技能 | 玩法含义 |
|------|------|----------|
| 1 | 弹窗雨 | 随机遮挡屏幕和碰撞区域 |
| 2 | 诱导按钮 | 场上出现高亮按钮，踩上去增伤但扣童心 |
| 3 | 试玩结束 | 倒计时后关闭部分技能，玩家需用琪琪的“继续试玩”打断 |

关键剧情：

- 琪琪不是“游戏公主”，而是童心界的第一个系统管理员。
- 她被困在一个永远关闭不了的窗口里，窗口每次刷新都会复制她的记忆。
- 星愿击败六元冠冕后，学会拒绝“快速变强”的诱导，保留自由玩的权利。

### 第二章：雪花放映厅

**核心记忆**：动画看到关键处时黑屏、雪花、缺集、色彩失真。  
**守护公主**：露露。  
**反派**：静屏馆长。  
**章节奖励**：胶片灯，解锁色彩通道机关和黑屏反制。

玩法机制：

- 地图有红、蓝、黄三种色彩通道，切换后会显现不同平台和敌人。
- 黑屏区域会暂时隐藏地形，玩家需要记忆路径或点亮胶片灯。
- “缺帧怪”会瞬移，只有在放映机光束中才显形。
- 玩家修复胶卷片段后，场景从黑白逐步恢复颜色。

BOSS 机制：

- 静屏馆长会关闭屏幕 2 秒，玩家只能根据音效和之前记忆躲避。
- 雪花弹幕会污染色彩通道，需要露露净化。
- 最终阶段只保留一个小放映窗口，玩家需要移动窗口照出 BOSS 弱点。

### 第三章：纸皮英雄基地

**核心记忆**：小时候相信披上披风就能变强，长大后却看见“英雄”被包装、续作和廉价特效耗空。  
**守护公主**：艾琳。  
**反派**：塑料特效王。  
**章节奖励**：星盾核心，强化护盾和救援机制。

玩法机制：

- 场景里有纸板城市、玩具楼房、投影爆炸，真假危险混在一起。
- 玩家要优先救出被困 NPC，救援成功会给“勇气槽”充能。
- 假英雄敌人只会摆姿势，真正威胁来自被忽视的场景事故。
- 艾琳加入后，玩家可用星盾反弹“假爆炸”。

BOSS 机制：

- 塑料特效王会制造巨大爆炸投影，只有投影边缘是真伤害。
- 场地上有“观众掌声槽”，玩家救人越多，BOSS 护盾越低。
- 最终阶段，BOSS 的特效全部断电，露出纸皮核心。

### 第四章：反面童话森林

**核心记忆**：童话里的希望被大人改写成“现实一点”的结局。  
**守护公主**：贝儿。  
**反派**：删改皇后。  
**章节奖励**：命运书签，解锁结局改写机关。

玩法机制：

- 地图由多个“故事页”组成，玩家在页与页之间移动。
- 每页都有一个被篡改的结局条款，例如“门永远打不开”“舞会永远迟到”。
- 玩家需要找到童话道具，把坏条款改成可游玩的规则。
- 选择并非单纯善恶，而是“相信一次”还是“提前放弃”。

BOSS 机制：

- 删改皇后会把玩家技能描述临时改坏，例如“火球变成烟花但不伤害”。
- 贝儿可用命运书签恢复一个被篡改的技能。
- 最终阶段，玩家需要在三个结局中选择一个主动承担代价的好结局。

### 第五章：铃声操场

**核心记忆**：课间、春游和暑假的自由被铃声、排名和试卷挤压。  
**守护公主**：小雪。  
**反派**：补课总监。  
**章节奖励**：下课铃，解锁时间暂停式短窗口机制。

玩法机制：

- 关卡在“上课”和“课间”两种状态之间切换。
- 上课状态下敌人变强、路线变窄、任务目标被锁。
- 课间状态只有 10 秒，玩家要快速完成探索、救人或机关。
- 错题怪被击败后不会消失，而是变成可踩的平台，象征“错误也能帮你前进”。

BOSS 机制：

- 补课总监召唤试卷墙、排名柱和红笔判定圈。
- 玩家不能只打 BOSS，还要抢回被锁住的“课间铃声”。
- 小雪觉醒后，能把一次失败判定改成“重做一遍”。

### 第六章：跑调童谣镇

**核心记忆**：熟悉旋律被噪音覆盖，童年歌声变成压力。  
**守护公主**：铃铛。  
**反派**：噪声公爵。  
**章节奖励**：节拍铃，解锁节奏桥和音符净化。

玩法机制：

- 地图道路会按节拍显现，踩错节拍会被推回。
- 噪音区会反转方向键或扰乱技能冷却显示。
- 玩家收集旋律碎片，恢复村庄不同区域的音乐层。
- 铃铛加入后，玩家可短暂让敌人进入“安静”状态。

BOSS 机制：

- 噪声公爵会改变背景节拍，玩家必须根据视觉脉冲行动。
- 场地分为高音、低音、鼓点三条轨道，站错轨道会被震退。
- 最终阶段用三段旋律同时净化 BOSS。

### 第七章：断格漫画城

**核心记忆**：热血故事停在最燃的一页，等待多年也没有结局。  
**守护公主**：美琪。  
**反派**：断更书记。  
**章节奖励**：分镜笔，解锁速度线移动和补完机关。

玩法机制：

- 城市被分成漫画分镜格，格子边框是墙。
- 玩家用速度线冲刺穿过未完成边框。
- 部分 NPC 被困在“回忆格”里，需要补完当前页才能前进。
- 收集墨水可把空白格画成桥、掩体或陷阱。

BOSS 机制：

- 断更书记会把战斗进度回滚到上一格。
- BOSS 召唤“未完待续”屏障，必须用美琪的分镜斩切开。
- 最终阶段玩家亲手画出最后一格，让 BOSS 失去拖延能力。

English: original black-and-white comic city level, top-down view, streets made of manga panel frames, halftone shadows and speed lines forming paths, a dark-haired princess swinging a glowing panel pen, blank panels being drawn into bridges, energetic comic mood with no existing character likenesses

### 第八章：问号实验室

**核心记忆**：好奇心本来允许失败，却被“标准答案”压成唯一选择。  
**守护公主**：诺诺。  
**反派**：标准答案博士。  
**章节奖励**：问号试管，解锁实验合成和非标准解法。

玩法机制：

- 地图机关由火、冰、电、风四类实验元素组成。
- 错误组合不会直接惩罚，而会生成临时地形或特殊敌人。
- 玩家需要用失败结果绕开标准路线。
- 诺诺加入后，可扫描机关，显示“可能但不唯一”的解法。

BOSS 机制：

- 标准答案博士会把场地分成 A/B/C/D 四个答案区。
- 站在唯一正确区只能免伤，站在创造性错误区可反击。
- 最终阶段玩家需要连续使用三个“错误实验”打破标准答案屏障。

English: original top-down 2D science laboratory, giant test tubes, gears, magnetic coils and floating question marks, a green princess holding a glowing test tube, failed experiments creating colorful temporary bridges, steampunk childhood science style, clear puzzle structure, no real scientist portraits, no text

### 隐藏章：回声音乐厅

**核心记忆**：真正想听见的声音被杂音、跑调和标准练习覆盖。  
**守护公主**：乐乐。  
**反派**：失谐琴师。  
**触发条件**：收集七个“半截旋律”，并在断格漫画城最终页选择“把歌听完”。  
**章节奖励**：金色指挥棒，增强所有公主羁绊技。

玩法机制：

- 场景由回声平台组成，玩家攻击会产生音波路径。
- 不同乐器代表不同移动规则：弦乐桥、鼓点跳台、管乐风洞。
- 玩家需要让十二件原创乐器回到正确声部。
- 乐乐加入后，能听见隐藏路线的回声。

### 终章：灰钟塔

**核心记忆**：长大不是敌人，但“只允许一种长大方式”会杀死童心。  
**最终反派**：灰钟局长。  
**终章奖励**：真结局与成年后重返童心层的能力。

场景结构：

| 区域 | 视觉变化 | 玩法 |
|------|----------|------|
| 塔外 | 童话城堡外壳逐渐变成写字楼 | 回顾前八章机制 |
| 塔腰 | 电梯、表格、打卡机、无尽走廊 | 时间循环与选择收束 |
| 塔顶 | 巨大灰钟和童心值显示器 | 终局 BOSS 战 |

BOSS 机制：

- 灰钟局长会把玩家已解锁机制逐个“标准化”，例如把分镜冲刺变成直线移动。
- 九位公主会依次恢复一个被标准化的机制。
- 当童心值降到 1 时，玩家不是靠数值爆发，而是选择“我会长大，但我不交出童心”。
- 最终攻击不是杀死成长，而是打碎灰钟局对成长的垄断。

真结局：

- 星愿回到现实层，发现六一已经过去。
- 他不再试图永远停在童年，但能在现实中保留童心层的入口。
- 秘密基地变成可随时回去的“内心房间”。
- 最后一幕：他把一颗星星糖放在小卖部门口，留给下一个快要看不见童心层的孩子。

## 4. 剧情到系统的映射

| 剧情概念 | 系统实现建议 | 当前可承接模块 |
|----------|--------------|----------------|
| 童心值 | 0-1000 长期资源，影响后处理、移动、对话和结局 | `EmotionSystem`、`PostProcess`、`SaveSystem` |
| 公主觉醒 | 全局 flag + 跟随状态 + 羁绊技解锁 | `Princess`、`BondTechnique`、`SaveSystem` |
| 领域章节 | 每章一个 `RegionId`，包含专属 tile、敌人、任务 | `RegionManager`、`MapRegion`、`QuestSystem` |
| BOSS 机制 | 每个 BOSS 至少一个独特规则，不只改血量 | `EnemyManager`、`Ability`、`ParticleSystem` |
| 基地回报 | 家具、玩具、谈心恢复童心 | `BuildingSystem`、`ToySystem`、`Inventory` |
| 选择诱导 | 短期收益换童心值损失 | `DialogueTree`、`QuestSystem` |
| 真结局 | 九位公主觉醒 + 童心值未断线 + 灰钟塔通关 | `SaveSystem` flags |

## 5. 叙事语气

文案要克制，不要把每句话都写成段子。建议三种语气轮换：

| 场景 | 语气 |
|------|------|
| 普通 NPC | 轻松、生活化、带一点自嘲 |
| 公主剧情 | 真诚、温暖、少说教 |
| 反派台词 | 像规则和压力，不像普通坏人 |

示例台词：

- 琪琪：“你可以输，但不能点那个写着‘立刻变强’的按钮。它会赢得很快，也会让你忘得很快。”
- 露露：“黑屏不可怕，可怕的是你以为故事从来没有播过。”
- 小雪：“错题不是怪物。把它改会了，它就变成路。”
- 美琪：“停在最燃的地方不算结局。今天我们把最后一格画出来。”
- 诺诺：“标准答案能让你过关，但问题本身会带你去更远的地方。”
- 灰钟局长：“你可以保留童年，但它必须符合格式。”
- 星愿：“我会长大。可我不想把颜色交出去。”

## 6. 提示词统一规范

所有图片、视频提示词都遵守以下规则：

1. 写“记忆体验”和“玩法可见元素”，不写现实作品名。
2. 明确输出用途：角色立绘、场景概念、BOSS 设计、游戏截图感、剧情 CG、宣传片镜头。
3. 明确构图：俯视角、横版概念图、角色半身、全身、16:9 过场等。
4. 明确机制可见：弹窗遮挡、黑屏光束、试卷墙、分镜桥、节拍格等。
5. 负向约束必须包含：无真实品牌、无版权角色、无可读文字、无 logo。

通用图片模板：

```text
中文：原创2D俯视角动作RPG{资产类型}，{主体}，{玩法机制可见}，{环境}，{色彩与光照}，清晰可玩布局，手绘游戏概念图，无真实品牌，无版权角色，无可读文字，无logo

English: original top-down 2D action RPG {asset type}, {subject}, visible {gameplay mechanic}, {environment}, {palette and lighting}, readable playable layout, hand-painted game concept art, no real brands, no copyrighted characters, no readable text, no logos
```

通用视频模板：

```text
中文：{6-10秒}游戏宣传片镜头，{镜头运动}，{角色动作}，{机制触发前后变化}，{情绪目标}，2D手绘动作RPG风格，动作清晰，无文字，无真实品牌，无版权角色

English: {6-10 second} game trailer shot, {camera movement}, {character action}, {before-and-after gameplay mechanic change}, {emotional goal}, 2D hand-painted action RPG style, clear readable action, no text, no real brands, no copyrighted characters
```

## 7. 垂直切片建议

优先做第一章完整试玩，而不是同时推进九章：

```text
序章
→ 弹窗游乐厅
→ 琪琪觉醒
→ 六元冠冕 BOSS
→ 回秘密基地摆放像素手柄
→ 童心值恢复
→ 存档读档
```

这一段必须验证：

- 怀旧记忆是否真的变成玩法。
- 玩家是否理解童心值。
- BOSS 是否有独特规则。
- 公主是否改变战斗或探索方式。
- 基地是否提供情绪和数值回报。
- 提示词生成的图像和视频是否能服务真实关卡，而不是只做气氛图。
