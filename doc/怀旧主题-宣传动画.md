# 《星愿之子》怀旧主题 — 宣传动画脚本

> 全部内容建立在 [怀旧主题-世界观与剧情](怀旧主题-世界观与剧情.md) 之上。
> 输出 **3 段宣传动画 + 1 段完整版 90 秒动画**，每段按 **Sora / Runway / Pika / MiniMax Video** 的格式提供提示词。

---

## 宣传片总体定位

- **主题**：童年 vs 长大，每个成年人都曾是孩子
- **情绪曲线**：温暖 → 燃 → 感动 → 治愈
- **总时长**：90 秒完整版（拆 3 段 ≤ 8s 短视频）
- **配乐**：[主题曲「星愿」](怀旧主题-音乐与音效.md#十五序章-op主题歌)
- **字幕语言**：中文（主）/ 英文（次）

### 第 0-10 秒：开篇 - 现实世界的灰色

- 街道上一个 12 岁的男孩（星愿）背着书包走在回家路上
