# 《星愿之子》怀旧主题 — 怪物 & BOSS 设计

> 全部内容建立在 [怀旧主题-世界观与剧情](怀旧主题-世界观与剧情.md) 之上。
> 怪物 = **童年阴影拟人化**；BOSS = **童年阴影的具象领主**。共 **30+ 普通怪 + 10 大 BOSS + 1 真终 BOSS**。

---

## 一、怪物设计总则

| 属性 | 设计 |
|------|------|
| 颜色基调 | 灰、紫、黑、暗绿（与童心界的暖色调形成对比） |
| 形状 | 棱角化、机械化、半透明化（暗示"虚假的成年人"） |
| 行动模式 | 单调、机械、缺乏童心（暗示"失去童心的人"） |
| 出现 | 在反派势力入侵的关卡 |
| 死亡 | 化为"失去的童年回忆"碎片，可拾取恢复童心值 |
